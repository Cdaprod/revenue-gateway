# Contributing

Please keep changes minimal and self-contained. All new code must include tests and documentation.

## Development
- Install dependencies: `npm install`
- Run tests: `npm test`

## Pull request checklist
- [ ] Code is self-contained and idempotent.
- [ ] No unnecessary new files or external dependencies.
- [ ] Tests added or updated as appropriate.
- [ ] Docs updated where needed.

## Commit message conventions
Use short, descriptive messages such as `feat: add handoff utilities` or `fix: correct plan mapping`.
